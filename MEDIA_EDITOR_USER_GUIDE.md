# Media Editor - Quick User Guide

## üé® How to Use the Crop Feature

### Step 1: Open Image Editor
- Tap camera/gallery icon in create post
- Select an image from your gallery
- Editor opens automatically

### Step 2: Activate Crop Mode
- Tap the **Crop** button at the bottom toolbar
- Image will show with:
  - Dark overlay outside crop area
  - White border around crop region
  - 8 white handles (4 corners + 4 edges)
  - Grid lines for composition

### Step 3: Adjust Crop Region
**Corner Handles (Big white circles with blue border):**
- Drag top-left handle ‚Üí Resize from that corner
- Drag top-right handle ‚Üí Resize from that corner
- Drag bottom-left handle ‚Üí Resize from that corner
- Drag bottom-right handle ‚Üí Resize from that corner

**Edge Handles (White bars on sides):**
- Drag top edge ‚Üí Adjust top boundary
- Drag bottom edge ‚Üí Adjust bottom boundary
- Drag left edge ‚Üí Adjust left boundary
- Drag right edge ‚Üí Adjust right boundary

### Step 4: Use Grid Lines
- 2 vertical lines + 2 horizontal lines
- Position your subject at intersection points
- Follow "rule of thirds" for better composition

### Step 5: Apply Crop
- Tap the **Done** button (‚úì) at top-right
- Cropped image added to your post
- Can re-edit anytime by tapping edit icon

## üåà How to Use Filters

### Step 1: Open Filter Panel
- In image editor, tap **Filter** button at bottom
- Horizontal scrollable panel appears above toolbar
- Shows 5 filter options

### Step 2: Preview Filters
**Available Filters:**
1. **Original** - No changes (default)
2. **Bright** - Enhanced brightness and clarity
3. **Vintage** - Warm, retro aesthetic
4. **B&W** - Black and white conversion
5. **Vivid** - Boosted colors and saturation

### Step 3: Apply Filter
- Tap any filter chip
- Loading spinner appears in button (processing)
- Image updates with filtered version (0.5-2 seconds)
- Selected filter highlighted in blue

### Step 4: Compare Filters
- Tap different filters to compare
- Switch back to "Original" anytime
- Choose the one that looks best

### Step 5: Save Filtered Image
- Tap **Done** button (‚úì) at top-right
- Filtered image added to your post

## üí° Pro Tips

### Cropping
- **Start wide, then narrow** - Begin with more area, then refine
- **Use grid lines** - Place subject at intersections for better composition
- **Corner handles for major changes** - Resize entire region
- **Edge handles for fine-tuning** - Adjust one side at a time
- **Minimum size** - Can't crop smaller than 50x50 pixels

### Filtering
- **Apply filters BEFORE cropping** - Easier to see full effect
- **Try multiple filters** - Quick to compare, find your favorite
- **Original is always available** - Can undo filter anytime
- **Combine with rotation** - Rotate ‚Üí Filter ‚Üí Crop for complete control

### Workflow Recommendations
**Portrait Photos:**
1. Rotate if needed (vertical orientation)
2. Apply filter for mood/style
3. Crop to focus on face
4. Tap Done

**Landscape Photos:**
1. Check orientation (horizontal)
2. Crop to remove unwanted edges
3. Apply vivid or bright filter
4. Tap Done

**Product Photos:**
1. Crop to remove background clutter
2. Apply bright filter for clarity
3. Center product using grid lines
4. Tap Done

**Artistic Photos:**
1. Apply vintage or B&W filter first
2. Crop for dramatic composition
3. Use grid for rule of thirds
4. Tap Done

## üö´ What's NOT Included (and Why)

### No Text Overlays
**Why?** Post screen already has caption field. Add text there instead.

**Benefits:**
- Less cluttered editor interface
- Faster editing experience
- Text is editable even after posting
- Consistent with other social apps

### No Drawing Tools
**Why?** Keep editor focused on essential photo editing.

**Alternative:** Use your phone's built-in photo editor for drawings, then import.

### No Stickers
**Why?** Prioritizing professional photo editing tools first.

**Future:** May add sticker library based on user feedback.

## ‚ùì Troubleshooting

### Crop handles not responding?
- Make sure you're in crop mode (Crop button highlighted blue)
- Try tapping the handle directly (30x30 pixel target)
- Restart editor by closing and reopening

### Filter not applying?
- Wait for loading spinner to disappear
- Check internet connection (images load from cache)
- Try a different filter, then switch back
- Close and reopen editor

### Image looks pixelated after editing?
- Original image may be low resolution
- Filters compress slightly for performance
- Try "Original" filter for highest quality
- Use high-resolution source images

### Crop region too small?
- Drag corner handles outward to expand
- Minimum size is 50x50 pixels
- Reset by exiting and re-entering crop mode

### Filter panel not showing?
- Make sure you're editing an IMAGE (not video)
- Filters only work on images
- Tap Filter button to toggle panel
- Try closing other panels first
